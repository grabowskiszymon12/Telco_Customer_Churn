# Telco Customer Churn Dashboard

Power BI dashboard analysing customer churn in a telecommunications company.

## Dashboard Overview

![Churn Overview](images/TelcoChurn.pdf)

## Key Insights

- Customers with month-to-month contracts have the highest churn rate.
- Electronic check users show a higher churn rate than customers using automatic payments.
- New customers are more likely to leave.
- Customers without technical support or online security have higher churn risk.
- Fiber optic customers show a relatively high churn rate.

## Dashboard Features

- Customer churn KPIs
- Churn analysis by contract type
- Churn analysis by payment method
- Churn analysis by internet service
- Customer tenure segmentation
- Monthly charges segmentation
- Interactive filters

## Files

- `TelcoChurn.pbix` – interactive Power BI report
- `telcoNotebook.ipynb` – data analysis and machine-learning notebook
